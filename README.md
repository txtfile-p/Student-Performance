Student Performance Analysis Dashboard:
This project is based on Student Performance Analysis using Power BI.
Features:
•	Data Cleaning using Power Query
•	Attendance Group creation using Conditional Column
•	KPI Cards
•	Pass Percentage calculation
•	Student Count by Department
•	Interactive slicer
•	Drill-through functionality
Tools Used:
•	Power BI
•	Power Query
•	DAX
Dashboard Overview
This dashboard helps analyze student performance, attendance distribution, and department-wise student details using interactive visualizations.
